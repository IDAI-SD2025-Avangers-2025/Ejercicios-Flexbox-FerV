Propiedades de flexbox:


    * display: flex;
    Hace que se active el flexbox, si no se usa correctamente, ninguna propiedad de flexbox funcionara
    *flex-direction
    Hace que exista una direccion en donde posicionar algo, ya sea en columnas o en filas
    *flex-wrap
    Determina la direccion de las lineas que se juntan
    *flex-flow
    Se especifica como se posicionan los bloques del flexbox
    *justify-content
    Posiciona el contenido en varios lugares, como el centro
    *align-content
    Es similar al justify-content, aunque es mas usado para elementos flex
    *align-items
    Alinea los items del flex en la linea del contenedor flex
    *align-self
    Permite que el alineamiento se use para items individuales, como el item 4 en el flexbox4
    *flex
    Utiliza el shrink, el grow, y el basis para modificar el cuadro
